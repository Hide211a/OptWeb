# ОптСклад — вимоги до системи

## Тема диплому

Розробка системи обліку та реалізації продовольчих товарів для оптової торгівлі.

## Призначення

Внутрішній веб-застосунок оптової бази продовольчих товарів: облік надходжень, залишків по партіях, реалізація (відвантаження) B2B-клієнтам, списання та звітність.

## Ролі

| Роль | Код | Можливості |
|------|-----|------------|
| Адміністратор | ADMIN | Довідники, користувачі, розпроведення |
| Менеджер складу | MANAGER | Документи, залишки, проведення |
| Керівник | DIRECTOR | Dashboard, звіти |

## Ключові функції

- Партійний облік з терміном придатності
- FEFO при реалізації та списанні
- Документи: надходження, реалізація, інвентаризація, списання
- Звіти та експорт CSV

## Стек

React + TypeScript + Vite + MUI | Node.js + Express + Prisma | SQLite
